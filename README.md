# DIY Crafts & Tutorials Website

This project helps beginners learn simple DIY crafts at home using everyday materials.

## Pages Included
1️⃣ Home Page – Website introduction  
2️⃣ Tutorials Page – Step-by-step instructions  
3️⃣ Materials Page – Table showing craft materials and uses  
4️⃣ Sign-Up Page – JavaScript validation included

• HTML  
• CSS  
• JavaScript

✔ Clean and simple UI  
✔ Navigation bar on all pages
